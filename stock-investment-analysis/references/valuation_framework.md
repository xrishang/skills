# 估值方法参考

本文档说明本 skill 用到的估值方法、公式、行业经验值。Claude 在做估值维度分析时参考本文档。

## 关键估值指标

### PE(TTM,市盈率)
**公式**:股价 / 每股收益 TTM,或 总市值 / 归母净利润 TTM
**含义**:买下整个公司需要多少年的净利润。低 = 便宜(或预期利润下滑),高 = 贵(或预期利润上涨)。
**A股经验值**(行业差异大,要和同行比):
- <10:低估,通常周期顶部、夕阳行业或短期业绩异常
- 10-20:合理偏低
- 20-30:合理偏高
- 30-50:高估,需要高成长性支撑
- >50:极度高估,或亏损(PE 失效)

### PB(市净率)
**公式**:股价 / 每股净资产,或 总市值 / 归母股东权益
**含义**:相对净资产的溢价。重资产行业看 PB,轻资产看 PE。
**经验值**:
- <1:破净,通常银行、地产、夕阳行业,或资产质量差
- 1-3:合理
- 3-5:偏高
- >5:高估,成长股或稀缺性资产

### PEG
**公式**:PE / 净利润增速(%)
**含义**:成长股专用。PEG < 1 视为便宜,PEG > 2 视为贵。
**适用**:有稳定高增长预期的成长股(如新能源、科技)
**不适用**:周期股、成熟股、亏损企业

### PS(市销率)
**公式**:总市值 / 营业收入
**含义**:对每元营收付多少钱。亏损或周期股用 PS 替代 PE。
**适用**:亏损企业、初创期互联网、电商
**经验值**:行业差异极大,主要和同行比

## 周期股的估值陷阱(重要)

周期股(钢铁、有色、煤炭、航运、化工)的 PE 是**反向指标**:
- **周期顶部**:利润最高,PE 看起来很低(5-8 倍),但实际是卖出时机
- **周期底部**:利润最低甚至亏损,PE 极高(50+ 倍)或失效,实际是买入时机

**周期股正确估值方法**:
1. 看 PB 分位,不看 PE 分位
2. 看产品价格周期位置(大宗商品价格在历史区间的位置)
3. 看行业产能利用率(高位是顶,低位是底)
4. 用 5-10 年 ROE 平均值代替当前 ROE

## 行业典型估值差异

| 行业 | 主用 | 经验值 | 原因 |
|---|---|---|---|
| 银行 | PB | 0.5-1.0 倍 | 杠杆高,利润受拨备调控 |
| 地产 | PB / NAV | 0.3-0.8 倍 | 资产价值真实,但去化周期影响大 |
| 周期股(钢煤化) | PB | 0.8-2.0 倍 | 利润波动大,看资产价值 |
| 消费(白酒食品) | PE | 20-40 倍 | 品牌溢价,盈利稳定 |
| 医药 | PE / PEG | 30-60 倍 | 成长性高,看 PEG |
| 科技(互联网) | PS / PE | 视阶段 | 早期看 PS,成熟看 PE |
| 半导体 | PB / PE | 30-100 倍 | 周期+成长,看产能和技术节点 |
| 新能源 | PEG / PE | 20-50 倍 | 成长性强,看渗透率 |
| 公用事业 | PB / 股息率 | 1-2 倍 | 现金流稳定,看股息率 |

## 历史分位的解读

本 skill 用股价历史分位作为估值分位的代理(因为拿不到 PE 历史数据)。注意:

- **股价分位 <20%**:大致对应"市场认为便宜",但可能是基本面恶化导致股价跌下来,需要交叉看财务维度
- **股价分位 20-80%**:中性
- **股价分位 >80%**:大致对应"市场认为贵",但可能是基本面大幅好转,需要交叉看成长性

## A股 vs 港股估值差异

- **港股普遍比 A股估值低 30-50%**:流动性折价、机构投资者占比高、外资定价
- **AH 股价差**:同一公司 A股和 H 股价格不同,通常 A股溢价 30-100%
- **港股科技股估值**:互联网公司用 PS 或 Non-IFRS 净利润 PE,不用 IFRS 净利润
- **港股财报术语差异**:港式记账用"溢利"代替"利润","营业额"代替"营业收入"

## 估值的局限性

估值是个**相对判断**,不是绝对真理:
1. 同样基本面,不同利率环境下估值合理区间不同(低利率时代估值普遍上移)
2. 估值便宜不等于会涨,需要催化剂(业绩、政策、事件)
3. 估值昂贵不等于会跌,高成长性可以消化高估值
4. 极端估值(破净 / 极高 PE)是强信号,但中间值(20-30 倍 PE)信息量有限
