# Skills · A 股 / 港股投资分析工具集

一套面向 Claude Code 的投资分析 skill 集合,覆盖 **全市场选股 → 个股深度分析 → ETF 资金流向分析** 的完整投研链路。每个 skill 都是独立可调用的模块,彼此也可组合使用:选股筛出的 Top N 可以直接交给个股分析做六维度深度复盘,ETF 分析与个股分析互为补充(ETF 看资金/行业,个股看财务/估值)。

## 适合谁用

- 个人投资者想做有依据的 A 股 / 港股决策,而不是凭感觉
- 需要快速从 5000+ 只股票中筛出候选标的
- 拿到一只股票后想要"值不值得买"的结构化判断
- 想知道资金正在往哪个行业/主题流入(ETF 维度)

## 包含的 Skills

### 1. `stock_screener/` — 全市场选股(广度)

从全市场 5000+ 只 A 股中按 4 种策略筛选 Top N:

| 策略 | 逻辑 | 适合风格 |
|------|------|----------|
| 技术面 | 趋势向上 + 放量 + 换手活跃 | 趋势跟踪 |
| 基本面 | 低 PE + 合理 PB + 大市值蓝筹 | 价值投资 |
| 资金面 | 主力净流入大 + 净占比强 | 跟随聪明钱 |
| 超跌反弹 | 60 日跌深 + 今日企稳 + 资金回流 | 抄底 |

输出 4 策略 Top N 候选 + 交叉验证,并对 Top 3 自动调用个股分析 skill 生成完整 PDF 报告。

### 2. `stock-investment-analysis/` — 个股深度分析(深度)

给一只 A 股或港股,从六个维度判断"现在是否值得投资",最终给出明确评级:

- **公司财务** — 盈利能力、成长性、偿债能力、现金流质量
- **估值** — 绝对估值(PE/PB/PEG/PS)+ 相对估值(历史分位、行业对比)
- **技术面** — 趋势、支撑压力位、量价配合、超买超卖
- **宏观经济** — 利率、汇率、CPI/PPI、政策周期
- **行业景气度** — 行业增长、竞争格局、政策方向、产业链位置
- **时事热点** — 近期新闻、业绩预告、突发事件

评级输出:**强烈推荐 / 推荐 / 观望 / 回避**,附理由与风险提示。六维交叉验证而非简单平均 —— 任何一个维度都可以独立否决。

### 3. `etf_analysis/` — ETF 资金流向分析(行业/主题维度)

给一个行业关键词(如"半导体")或"全市场 Top N"指令,从四个维度分析 ETF:

- **资金面** — 主力净流入 / 超大单 / 大单 / 中单 / 小单,谁在买
- **流动性** — 规模、成交额、换手率,能不能买得动
- **技术面** — MA / MACD / RSI / KDJ,价格位置
- **行业面** — 跟踪指数、行业政策、景气度

与个股分析的根本区别:ETF 不看公司财务(因为没有公司),核心是看**资金在往哪个行业/主题流入**。

## 工作流(三 skill 如何配合)

```
                    全市场 5000+ A 股
                          │
                 ┌────────▼────────┐
                 │  stock_screener │  4 策略筛选 Top N
                 └────────┬────────┘
                          │
              ┌───────────┴───────────┐
              │                       │
     Top 3 个股深度分析          其余候选观察
              │
     ┌────────▼────────┐
     │stock-investment-│  六维度评级
     │    analysis     │  强烈推荐/推荐/观望/回避
     └─────────────────┘

并行:行业维度
     ┌─────────────────┐
     │   etf_analysis  │  资金流向排行
     └─────────────────┘
```

选股负责"广度扫描",个股分析负责"深度判断",ETF 分析负责"行业/主题资金面"。三者既可独立使用,也可组合成完整投研闭环。

## 通用目录结构

每个 skill 内部组织一致:

```
<skill>/
├── SKILL.md          # skill 说明(给 Claude 读的指令)
├── scripts/          # 抓数据 / 算指标 / 画图 / 生成 PDF 的脚本
├── assets/           # 报告模板
├── references/       # 框架文档(估值/评级/策略等)
└── output/           # 生成的报告(PDF/Markdown/CSV/PNG)
```

## 数据源

通过 `akshare` 等 Python 库抓取 A 股 / 港股的行情、财报、资金流向、宏观经济数据,所有脚本可本地运行。
