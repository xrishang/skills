# 选股策略补充说明

## 4 种策略的适用场景

### 技术面选股
- **适合**:牛市或震荡上行市,趋势跟踪
- **失效**:熊市或震荡下行,趋势反转频繁
- **陷阱**:涨高的票追高风险大,需结合量能确认(放量突破 vs 放量出货)

### 基本面选股
- **适合**:震荡市或熊市,价值投资
- **失效**:题材炒作市(低 PE 跑不赢高成长)
- **陷阱**:周期股顶部 PE 低(利润最高时),周期股底部 PE 高(利润最低时)。需结合行业维度判断周期位置

### 资金面选股
- **适合**:资金驱动的行情,主力资金明确流入方向时
- **失效**:散户主导的小盘股行情(主力数据噪音大)
- **陷阱**:单日净流入可能是"一日游",需观察 3-5 日持续性

### 超跌反弹
- **适合**:急跌后的反弹行情,或恐慌底之后
- **失效**:趋势性熊市(超跌可能继续超跌)
- **陷阱**:基本面恶化的股票(低 PE 可能是价值陷阱),超跌反弹后可能继续跌

## 交叉验证的价值

单策略筛选有盲区,多策略交叉验证更可靠:
- **fundamental + fund_flow**:低估值 + 主力加仓 = 价值资金共振(最可靠)
- **technical + fund_flow**:趋势 + 资金 = 强势股确认(适合追涨)
- **oversold + fund_flow**:超跌 + 资金回流 = 抄底信号(需要技术面止跌确认)
- **technical + fundamental**:趋势 + 低估值 = 戴维斯双击(理想组合)

## 筛选条件的调参建议

当前筛选条件是"中等严格"的默认值,用户可根据需要调整:

### 技术面
- 激进版:60日涨跌幅 >20% + 今日涨幅 >5% + 量比 >2
- 保守版:60日涨跌幅 >5% + 今日涨幅 >1% + 量比 >1

### 基本面
- 激进版(深度价值):PE 5-15 + PB 0.5-2
- 保守版(合理估值):PE 15-40 + PB 2-8

### 资金面
- 激进版:净流入 >1亿 + 净占比 >10%
- 保守版:净流入 >2000万 + 净占比 >3%

### 超跌
- 激进版(深跌):60日涨跌幅 <-40%
- 保守版(浅跌):60日涨跌幅 <-10%
